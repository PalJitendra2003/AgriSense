🌱 AgriSense – AI-Powered Crop Disease Detection System

AgriSense is an AI-based agricultural application designed to detect crop diseases from plant leaf images and provide actionable treatment recommendations. The system helps farmers identify diseases early, reduce crop loss, and improve agricultural productivity.

🚀 Live Demo

🔗 https://agrisensee.vercel.app

🧠 Features
🌿 Crop disease detection using deep learning
📸 Image-based plant analysis
💊 Disease treatment recommendations
⚡ Fast and user-friendly web interface
🌍 Real-world application for smart agriculture
